# Table App

Live: https://table.f.avds.dev/

## How to run in development mode

### Clone repository

```
git clone https://github.com/souavds/f.git && cd f/table
```

### Install dependencies

```
npm install
```

### Run the project

```
npm start
```

## How to test the application

```
npm test
```

## How to build the application for production

```
npm build
```
